void SCRadixPrintTree ( SCRadixTree *  tree)

Prints the Radix Tree. While printing the radix tree we use the following format. 

   Parent_0
       Left_Child_1
           Left_Child_2
           Right_Child_2
       Right_Child_1
           Left_Child_2
           Right_Child_2     and so on

   Each node printed out holds details on the next bit that differs
   amongst its children, and if the node holds a prefix, the perfix is
   printed as well.
Parameters
treePointer to the Radix tree that has to be printed
