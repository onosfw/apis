int GetIfaceOffloading ( const char *  pcap_dev)

output offloading status of the link

Test interface for GRO and LRO features. If one of them is activated then suricata mays received packets merge at reception. The result is oversized packets and this may cause some serious problem in some capture mode where the size of the packet is limited (AF_PACKET in V2 more for example).
