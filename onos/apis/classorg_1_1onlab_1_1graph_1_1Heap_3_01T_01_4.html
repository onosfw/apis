Detailed Description

Implementation of an array-backed heap structure whose sense of order is imposed by the provided comparator.

While this provides similar functionality to java.util.PriorityQueue data structure, one key difference is that external entities can control when to restore the heap property, which is done through invocation of the heapify method.

This class is not thread-safe and care must be taken to prevent concurrent modifications.
