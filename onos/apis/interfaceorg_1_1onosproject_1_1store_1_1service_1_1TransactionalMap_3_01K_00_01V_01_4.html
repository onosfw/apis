Detailed Description

Transactional Map data structure.

A TransactionalMap is created by invoking getTransactionalMap method. All operations performed on this map within a transaction boundary are invisible externally until the point when the transaction commits. A commit usually succeeds in the absence of conflicts.

Parameters
<K>type of key.
<V>type of value.
