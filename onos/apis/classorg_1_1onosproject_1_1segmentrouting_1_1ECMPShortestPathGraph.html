Detailed Description

This class creates bandwidth constrained breadth first tree and returns paths from root Device to leaf Devices which satisfies the bandwidth condition.

If bandwidth parameter is not specified, the normal breadth first tree will be calculated. The paths are snapshot paths at the point of the class instantiation.
