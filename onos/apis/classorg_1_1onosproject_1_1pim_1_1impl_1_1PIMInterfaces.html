Detailed Description

PIMInterfaces is a collection of all neighbors we have received PIM hello messages from.

The main structure is a HashMap indexed by ConnectPoint with another HashMap indexed on the PIM neighbors IPAddress, it contains all PIM neighbors attached on that ConnectPoint.
