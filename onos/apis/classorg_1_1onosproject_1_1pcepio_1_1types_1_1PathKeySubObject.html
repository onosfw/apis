Detailed Description

Path Key SubObject: When a PCC needs to expand a path-key in order to expand a CPS, it issues a Path Computation Request (PCReq) to the PCE identified in the PKS in the RSVP-TE ERO that it is processing.

The PCC supplies the PKS to be expanded in a PATH-KEY SubObject in the PCReq message.
