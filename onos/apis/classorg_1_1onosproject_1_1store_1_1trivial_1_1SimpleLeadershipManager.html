Detailed Description

A trivial implementation of the leadership service.

The service is not distributed, so it can assume there's a single leadership contender. This contender is always granted leadership whenever it asks.

boolean org.onosproject.store.trivial.SimpleLeadershipManager.stepdown ( String  path)

If the local nodeId is the leader for specified topic, this method causes it to step down temporarily from leadership.

The node will continue to be in contention for leadership and can potentially become the leader again if and when it becomes the highest priority candidate

If the local nodeId is not the leader, this method will make no changes and simply return false.

Parameters
pathtopic for which this controller node should give up leadership
Returns
true if this node stepped down from leadership, false otherwise
