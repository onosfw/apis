Detailed Description

Represents for a order that consumer subscribe tunnel.

ONOS maintains request information, it means ONOS knows how much resource echo consumer uses in the ONOS. Although there is no a tunnel that consumer want to use, when producer creates a new tunnel, ONOS will notify the consumers that want to use it.
