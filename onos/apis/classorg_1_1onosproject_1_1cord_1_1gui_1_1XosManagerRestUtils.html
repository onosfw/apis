org.onosproject.cord.gui.XosManagerRestUtils.XosManagerRestUtils ( String  xosServerAddress,
int  xosServerPort,
String  baseUri 
)

Constructs a utility class, using the supplied server address and port, using the given base URI.

Note that the uri should start and end with a slash; for example:

"/volttenant/"

. This example would result in URIs of the form: 

    "http://[server]:[port]/xoslib/volttenant/"
Parameters
xosServerAddressserver IP address
xosServerPortserver port
baseUribase URI
