Detailed Description

Provides manually scheduled timer utility.

All schedulable methods are subject to overflow (you can set a period of max long). Additionally if a skip skips a period of time greater than one period for a periodic task that task will only be executed once for that skip and scheduled it's period after the last execution.

Returns the new simulated current time in millis after advancing the absolute value of millis to advance.
