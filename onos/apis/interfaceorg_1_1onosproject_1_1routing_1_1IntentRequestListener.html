void org.onosproject.routing.IntentRequestListener.updateExistingMp2pIntent ( IpPrefix  ipPrefix,
ConnectPoint  ingressConnectPoint 
)

Adds one new ingress connect point into ingress points of an existing intent and resubmits the new intent.

If there is already an intent for an IP prefix in the system, we do not need to create a new one, we only need to update this existing intent by adding more ingress points.

Parameters
ipPrefixthe IP prefix used to search the existing MultiPointToSinglePointIntent
ingressConnectPointthe ingress connect point to be added into the exiting intent
