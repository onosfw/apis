Detailed Description

Implementation of ODU (Optical channel Data Unit) signal ID signal criterion.

This criterion is based on the specification of "OFPXMT_EXP_ODU_SIGID" in Open Networking Foundation "Optical Transport Protocol Extension Version 1.0", but defined in protocol agnostic way.
