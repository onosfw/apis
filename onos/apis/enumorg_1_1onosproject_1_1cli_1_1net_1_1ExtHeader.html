static

Parse a string input that could contain an ExtHeader value.

The value may appear in the string either as a known exntension header name (one of the values of this enum), or a numeric extension header value.

Parameters
inputthe input string to parse
Returns
the numeric value of the parsed IPv6 extension header
Exceptions
