static String org.onosproject.routing.RouteEntry.createBinaryString ( IpPrefix  ipPrefix)
static

Creates the binary string representation of an IP prefix.

The prefix can be either IPv4 or IPv6. The string length is equal to the prefix length + 1.

For each string, we put a extra "0" in the front. The purpose of doing this is to store the default route inside InvertedRadixTree.

ipPrefixthe IP prefix to use

the binary string representation
