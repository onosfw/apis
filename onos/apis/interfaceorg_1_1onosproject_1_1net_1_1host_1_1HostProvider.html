void org.onosproject.net.host.HostProvider.triggerProbe ( Host  host)

Triggers an asynchronous probe of the specified host, intended to determine whether the host is present or not.

An indirect result of this should be invocation of org.onosproject.net.host.HostProviderService#hostDetected or org.onosproject.net.host.HostProviderService#hostVanished at some later point in time.
