Constructs a table request handler for a specific table view.

When table requests come in, the handler will generate the appropriate table rows, sort them according the the request sort parameters, and send back the response to the client.

Parameters
reqTypetype of the request event
respTypetype of the response event
nodeNamename of JSON node holding row data
