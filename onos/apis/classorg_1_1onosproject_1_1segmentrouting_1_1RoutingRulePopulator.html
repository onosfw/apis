void org.onosproject.segmentrouting.RoutingRulePopulator.populateRouterMacVlanFilters ( DeviceId  deviceId)

Creates a filtering objective to permit all untagged packets with a dstMac corresponding to the router's MAC address.

For those pipelines that need to internally assign vlans to untagged packets, this method provides per-subnet vlan-ids as metadata.

Note that the vlan assignment is only done by the master-instance for a switch. However we send the filtering objective from slave-instances as well, so that drivers can obtain other information (like Router MAC and IP).

Parameters
deviceIdthe switch dpid for the router
