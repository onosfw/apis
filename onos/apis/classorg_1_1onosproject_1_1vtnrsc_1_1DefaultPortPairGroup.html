boolean org.onosproject.vtnrsc.DefaultPortPairGroup.exactMatch ( PortPairGroup  portPairGroup)

Returns whether this port pair group is an exact match to the port pair group given in the argument.

Exact match means the Port pairs match with the given port pair group. It does not consider the port pair group id, name and description.

Parameters
portPairGroupother port pair group to match against
Returns
true if the port pairs are an exact match, otherwise false
